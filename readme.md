
# It's Bartini Time!

**************************************************

A simple web application to look up cocktail recipes using 
APIs from [TheCocktailDB](https://www.thecocktaildb.com/).

**************************************************

## Technologies Used

**************************************************

- HTML5
- CSS3
- JavaScript
- jQuery
- Google Fonts
- AJAX
- TheCocktailDB APIs
